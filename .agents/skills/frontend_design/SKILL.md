# Frontend Design Guidelines

## Aesthetics
- **Modern Look**: Use a premium, sleek design with vibrant colors.
- **Themes**: Implement a polished Dark Mode by default.
- **Typography**: Utilize modern Google Fonts like Inter, Roboto, or Outfit.
- **Visual Cues**: Add micro-animations and smooth transition effects to enhance the user experience.

## Layout & Components
- Build reusable and focused components.
- Ensure the interface is responsive across devices (especially desktop for simulator dashboards).
- Use data tables effectively with clear sorting and pagination for viewing statistics.

## Interactive Elements
- Provide visual feedback during simulations (e.g., loading spinners, progress bars).
- Clear action buttons for "Simulate Next Round" or "Simulate Season".
