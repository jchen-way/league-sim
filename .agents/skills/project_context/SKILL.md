# Project Context: Game Simulator

This project is a game simulator application.

## Goals
- Allow users to create and manage teams.
- Manage seasons and rounds.
- Track detailed statistics per season and cumulatively.
- Simulate matches/rounds between teams individually or all at once.
- Export data to CSV/Excel formats for external analysis.

## Key Concepts
- **Team**: The core entity that participates in matches.
- **Season**: A collection of rounds in which teams compete.
- **Round**: A specific set of matches played at one time.
- **Statistics**: Performance metrics recorded during simulations.
