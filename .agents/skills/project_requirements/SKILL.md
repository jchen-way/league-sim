# Project Requirements

## Core Features
1. **Team Management**: Create, edit, and delete teams.
2. **Season Management**: Start new seasons, track current season, view past seasons.
3. **Simulation Engine**: 
   - Simulate a single round.
   - Simulate all remaining rounds in a season.
   - The simulation results should affect team statistics.
4. **Statistics Tracking**: Track per-season stats and cumulative (all-time) stats for each team.
5. **Data Export**: Export team and season data to CSV/Excel.

## Non-Functional Requirements
- Ensure smooth simulation performance.
- Maintain accurate and reliable state management.
- Provide clear feedback to the user upon simulating rounds.
