# Technology Stack

## Frontend
- **Framework**: Next.js (React) or Vite.
- **Styling**: Vanilla CSS or Tailwind CSS (only if explicitly requested).
- **State Management**: React Context or lightweight libraries like Zustand.

## Backend
- **Runtime**: Node.js.
- **Database**: PostgreSQL with Prisma ORM or SQLite.
- **API**: REST or GraphQL.

## Other Tools
- **Export**: `xlsx` or `csv-writer` for data export.
